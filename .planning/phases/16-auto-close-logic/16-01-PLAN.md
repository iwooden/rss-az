```cython
from core.state cimport GameState
```

**Helper 1: `_close_company()`**
- Parameters: `GameState state, int company_id, int owner_type, int owner_id`
- Clear ownership from previous owner (FI or corp)
- Apply Junkyard Scrappers bonus: if JS (corp_id 0) is active, add 2x printed income to JS cash
- Call `company_module.COMPANIES[company_id].remove_from_game(state)`

**Helper 2: `_process_fi_auto_close()`**
- Get current CoO level from `turn_module.TURN.get_coo_level(state)`
- Iterate all 36 companies
- For each company owned by FI:
  - Get base income via `get_company_income(company_id)`
  - Get stars via `get_company_stars(company_id)`
  - Get CoO value via `get_cost_of_ownership(coo_level, stars)`
  - If `base_income - coo < 0` (NEGATIVE adjusted income, not <= 0): close company

**Helper 3: `_process_receivership_auto_close()`**
- Iterate all 8 corps
- Skip inactive corps and non-receivership corps
- For each receivership corp:
  - Check if corp is Vintage Machinery (corp_id 6) for CoO reduction
  - Find highest face value company (protected) - iterate owned companies, track max
  - Iterate owned companies again (excluding protected company):
    - Get stars and CoO value
    - Apply VM reduction: if is_vm, `coo = max(0, coo - 10)`
    - Red (stars=1): close if CoO >= 4
    - Orange (stars=2): close if CoO >= 7
    - Yellow/Green/Blue (stars 3-5): NEVER auto-close

**Main entry: `apply_closing_auto()`**
```cython
cdef int apply_closing_auto(GameState state) noexcept:
    """Execute auto-close logic for FI and receivership corps.

    This is a deterministic non-player phase with 0 actions.
    Returns: 0 always (deterministic, no failure modes)
    """
    _process_fi_auto_close(state)
    _process_receivership_auto_close(state)
    return 0
```

**Python wrapper for testing:**
