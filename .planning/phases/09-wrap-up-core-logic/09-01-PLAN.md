Create wrap_up.pyx following bid.pyx pattern with:
1. Standard Cython directives (language_level=3, boundscheck=False, wraparound=False, cdivision=True)
2. Imports: GameState, GamePhases, turn_module, player_module
3. Helper `_reorder_players_by_cash(GameState state)`:
   - Declare all cdef vars at function start (num_players, cash_values[6], old_positions[6], player_ids[6], new_order[6], loop vars)
   - Gather current cash and turn_order for each player using PLAYERS[i].get_cash and get_turn_order
   - Selection sort by (-cash, old_position): higher cash wins, equal cash -> lower old position wins
   - Apply new turn orders using PLAYERS[player_ids[i]].set_turn_order(state, i)
   - Set active player to player_ids[0] using state._set_active_player()

4. Main handler `apply_wrap_up(GameState state) noexcept`:
   - Call _reorder_players_by_cash(state)
   - Clear consecutive passes: turn_module.TURN.clear_consecutive_passes(state)
   - Transition to ACQUISITION: turn_module.TURN.set_phase(state, GamePhases.PHASE_ACQUISITION)
   - Return 0

Important: WRAP_UP does NOT take ActionInfo* since it has no actions. Different signature from bid/invest handlers.
