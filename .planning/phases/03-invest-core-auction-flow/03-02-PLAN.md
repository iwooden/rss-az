**Helper functions to add:**

1. `_find_player_at_position(state, position)` - Find player_id with given turn_order position
2. `_count_active_bidders(state)` - Count players who haven't left auction
3. `_advance_to_next_bidder(state)` - Advance to next non-passed bidder in turn order
4. `_set_active_player_after(state, player_id)` - Set active to next after given player
5. `_resolve_auction(state)` - Complete auction resolution sequence

**Leave auction action (ACTION_LEAVE_AUCTION - BID-01, BID-02):**
1. Get current player: `state._get_active_player()`
2. Set passed: `turn_module.TURN.set_player_passed_auction(state, player_id, True)`
3. Count remaining bidders: `_count_active_bidders(state)`
4. If count == 1: `_resolve_auction(state)` - auction over
5. Else: `_advance_to_next_bidder(state)` - continue auction
6. Return 0

**Raise bid action (ACTION_RAISE_BID - BID-03, BID-04):**
1. Get company: `turn_module.TURN.get_auction_company(state)`
2. Calculate new bid: `get_company_face_value(company_id) + info.amount + 1`
   (info.amount is 0-18, representing bids from face+1 to face+19)
3. Get player_id: `state._get_active_player()`
4. Update auction state:
   - `TURN.set_auction_price(state, new_bid)`
   - `TURN.set_auction_high_bidder(state, player_id)`
5. Advance to next bidder: `_advance_to_next_bidder(state)`
6. Return 0

**Auction resolution function (_resolve_auction - BID-05 through BID-12):**
1. Get winner: `turn_module.TURN.get_auction_high_bidder(state)`
2. Get starter: `turn_module.TURN.get_auction_starter(state)`
3. Get company: `turn_module.TURN.get_auction_company(state)`
4. Get price: `turn_module.TURN.get_auction_price(state)`
5. Winner pays (BID-06): `player_module.PLAYERS[winner_id].add_cash(state, -price)`
6. Winner receives company (BID-07): `company_module.COMPANIES[company_id].transfer_to_player(state, winner_id)`
7. Update net worth (BID-12): `player_module.PLAYERS[winner_id].update_net_worth(state)`
8. Draw new company (BID-09):
   ```cython
   cdef int new_company = deck_module.DECK.draw(state)
   if new_company >= 0:
       company_module.COMPANIES[new_company].move_to_auction(state)
   ```
9. Clear auction state (BID-08):
   - `TURN.clear_auction_company(state)`
   - `TURN.clear_auction_high_bidder(state)`
   - `TURN.clear_auction_starter(state)`
   - `TURN.clear_auction_passed(state)`
   - `TURN.set_auction_price(state, -1)`
10. Return to INVEST (BID-10): `TURN.set_phase(state, GamePhases.PHASE_INVEST)`
11. Next player after starter (BID-11): `_set_active_player_after(state, starter_id)`

**Important patterns from research:**
- Count active bidders by checking `has_player_passed_auction(state, player_id)` for all players
- Bidder rotation follows turn order, not player_id order
- Winner is NOT necessarily next player - turn goes to player after STARTER
