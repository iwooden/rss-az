**Helper functions to add:**

1. `_find_player_at_position(state, position)` - Find player_id with given turn_order position
2. `_advance_active_player(state)` - Advance to next player in turn order
3. `_advance_to_next_bidder(state)` - Advance to next non-passed bidder (used after auction start)

**Pass action (ACTION_PASS):**
1. Increment consecutive_passes via `turn_module.TURN.increment_consecutive_passes(state)`
2. Check if `get_consecutive_passes(state) >= state._num_players`
3. If yes: transition to WRAP_UP via `set_phase(state, GamePhases.PHASE_WRAP_UP)`
4. If no: advance to next player via `_advance_active_player(state)`
5. Return 0

**Auction action (ACTION_AUCTION):**
1. Get company_id from slot via `get_auction_company_for_slot(state, info.slot)`
2. Calculate bid price: `get_company_face_value(company_id) + info.amount`
3. Get starter player_id: `state._get_active_player()`
4. Set auction state:
   - `TURN.set_auction_company(state, company_id)`
   - `TURN.set_auction_price(state, bid_price)`
   - `TURN.set_auction_high_bidder(state, player_id)`
   - `TURN.set_auction_starter(state, player_id)`
   - `TURN.clear_auction_passed(state)`
5. Clear consecutive passes: `TURN.clear_consecutive_passes(state)` (INV-02)
6. Transition phase: `TURN.set_phase(state, GamePhases.PHASE_BID_IN_AUCTION)`
7. Advance to next bidder (skipping passed players): `_advance_to_next_bidder(state)`
8. Return 0

**Buy/Sell share stubs remain unchanged** - implemented in Phase 4.
