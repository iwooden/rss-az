<task type="auto">
  <name>Task 2: Implement action handlers</name>
  <files>phases/acquisition.pyx</files>
  <action>
Add action handler functions in a new section "# ACTION HANDLERS" after validation helpers:

1. **_handle_accept_price(state, price)** - void noexcept:
   - Get corp_id (buyer) from TURN.get_acq_active_corp
   - Get company_id (target) from TURN.get_acq_target_company
   - Determine seller from company location:
     - Get company handle: company_module.COMPANIES[company_id]
     - Get location: company.get_location(state)
     - If LOC_CORP (2): seller_corp_id = company.get_owner_id(state)
     - If LOC_PLAYER (1): seller_player_id = company.get_owner_id(state)
   - Transfer money (buyer pays):
     - corp_module.CORPS[CORP_NAMES[corp_id]].add_cash(state, -price)
   - Seller receives (to acquisition_proceeds):
     - If corp seller: corp_module.CORPS[CORP_NAMES[seller_corp_id]].add_acquisition_proceeds(state, price)
     - If player seller: player_module.PLAYERS[seller_player_id].add_acquisition_proceeds(state, price)
   - Transfer company to buyer's acquisition zone:
     - company.transfer_to_corp_acquisition(state, corp_id)
   - Advance to next offer:
     - _advance_to_next_offer(state)

2. **_handle_fi_buy_high(state)** - void noexcept:
   - Get corp_id and company_id from TURN state
   - Get high_price = company_module.COMPANIES[company_id].get_high_price()
   - Transfer money: corp.add_cash(-high_price), fi_module.FI.add_cash(high_price)
   - Transfer company: company.transfer_to_corp_acquisition(state, corp_id)
   - Advance: _advance_to_next_offer(state)

3. **_handle_fi_buy_face(state)** - void noexcept:
   - Get corp_id and company_id from TURN state
   - Get face_value = get_company_face_value(company_id)
   - Transfer money: corp.add_cash(-face_value), fi_module.FI.add_cash(face_value)
   - Transfer company: company.transfer_to_corp_acquisition(state, corp_id)
   - Advance: _advance_to_next_offer(state)

4. **_handle_pass(state)** - void noexcept:
   - Simply call _advance_to_next_offer(state)
   - Pass permanently skips current offer (index advances, never re-offered)

Company location constants (define if not present):
- DEF LOC_FI = 0
- DEF LOC_PLAYER = 1
- DEF LOC_CORP = 2
