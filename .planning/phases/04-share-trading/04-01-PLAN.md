Add two cpdef helper methods to the Market class for finding next available market spaces:

1. `cpdef int find_next_higher_space(self, GameState state, int current_index)`:
   - Start at current_index + 1
   - Loop through market spaces until finding one where `state._data[self._market_offset + index] == 1.0`
   - Stop before index 26 (price 75) which is always available
   - If no available space found before 26, return 26 (price 75 is always valid target)
   - Return the found index

2. `cpdef int find_next_lower_space(self, GameState state, int current_index)`:
   - Start at current_index - 1
   - Loop downward through market spaces until finding one where `state._data[self._market_offset + index] == 1.0`
   - If no available space found (all occupied), return 0 (bankruptcy)
   - Return the found index

Import GameConstants for NUM_MARKET_SPACES (27).
