<task type="auto">
  <name>Task 3: Pre-allocate mask buffer instead of np.zeros() per call</name>
  <files>core/actions.pyx, core/actions.pxd</files>
  <action>
Pre-allocate a mask buffer to avoid `np.zeros()` allocation every call.

Strategy: Use a module-level buffer and clear it with memset. This is safe because:
- Python GIL ensures single-threaded access
- Buffer is cleared before each use

Add to actions.pxd:
```cython
# Pre-allocated mask buffer (max size for 6 players = 306)
cdef float _mask_buffer[306]
```

In actions.pyx, add at module level (after imports):
```cython
from libc.string cimport memset

# Maximum action count (6 players)
DEF MAX_ACTION_COUNT = 306

# Module-level mask buffer
cdef float _mask_buffer[MAX_ACTION_COUNT]
```

Update `get_valid_action_mask()`:
```cython
cpdef object get_valid_action_mask(GameState state):
    cdef int num_players = state._num_players
    cdef ActionLayout layout = compute_action_layout(num_players)
    cdef int total_actions = layout.total_size

    # Clear buffer (faster than np.zeros allocation)
    memset(_mask_buffer, 0, total_actions * sizeof(float))

    cdef int phase = state.get_phase()
    _fill_mask_for_phase(state, phase, &layout, _mask_buffer)

    # Copy to numpy array for return (required for Python interface)
    cdef cnp.ndarray mask = np.empty(total_actions, dtype=np.float32)
    cdef float* mask_ptr = <float*>cnp.PyArray_DATA(mask)
    cdef int i
    for i in range(total_actions):
        mask_ptr[i] = _mask_buffer[i]

    return mask
```

Similarly update `get_forced_action()` to use `_mask_buffer` instead of creating new numpy array.

Note: The copy to numpy is still needed for the Python return value, but internal operations use the static buffer.
