Create `entities/encoding.pyx` implementing the three one-hot encoding helpers.

Implementation details:
```cython
# cython: language_level=3, boundscheck=False, wraparound=False, cdivision=True

cdef inline void set_one_hot(float* data, int offset, int size, int value) noexcept nogil:
    """Set one-hot encoding: clear all, then set value position to 1.0."""
    cdef int i
    for i in range(size):
        data[offset + i] = 0.0
    if 0 <= value < size:
        data[offset + value] = 1.0

cdef inline int get_one_hot_index(float* data, int offset, int size) noexcept nogil:
    """Get index of 1.0 in one-hot encoding. Returns -1 if not found."""
    cdef int i
    for i in range(size):
        if data[offset + i] == 1.0:
            return i
    return -1

cdef inline void clear_one_hot(float* data, int offset, int size) noexcept nogil:
    """Clear one-hot encoding (set all to 0.0)."""
    cdef int i
    for i in range(size):
        data[offset + i] = 0.0
```

Note: These functions take raw float* pointer, not GameState, so they can be used in any context including nogil blocks.
