```cython
from core.state cimport GameState

# Status codes for apply_action return
cdef enum ActionStatus:
    STATUS_OK = 0           # Action applied successfully
    STATUS_INVALID = 1      # Invalid action for current state
    STATUS_GAME_OVER = 2    # Game ended after this action

cdef class GameDriver:
    cpdef int apply_action(self, GameState state, int action_idx)
    cpdef object get_legal_moves(self, GameState state)
```

**core/driver.pyx:**
- `cdef class GameDriver` with `__cinit__` (empty, no state needed)
- `cpdef int apply_action(self, GameState state, int action_idx)`:
  1. Get action mask via `get_valid_action_mask(state)`
  2. Validate action_idx is in bounds and mask[action_idx] == 1.0
  3. Return STATUS_INVALID if validation fails
  4. Decode action via `compute_action_layout()` and `decode_action()`
  5. Dispatch based on `state.get_phase()`:
     - PHASE_INVEST -> `apply_invest_action(state, &info)`
     - PHASE_BID_IN_AUCTION -> `apply_bid_action(state, &info)`
     - Other phases -> return STATUS_INVALID (stubs for later)
  6. Check if `state.get_phase() == PHASE_GAME_OVER` after dispatch, return STATUS_GAME_OVER if so
  7. Return result from phase handler
- `cpdef object get_legal_moves(self, GameState state)`:
  - Simply return `get_valid_action_mask(state)` (wraps existing function)
- Create global singleton: `DRIVER = GameDriver()`
